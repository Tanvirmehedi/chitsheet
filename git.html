Git CommentLine
Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency. Git is easy to learn and has a tiny footprint with lightning fast performance. It outclasses SCM tools like Subversion, CVS, Perforce, and ClearCase with features like cheap local branching, convenient staging areas, and multiple workflows.

Git initialization
use new project in git repository

//git initialization
git init

//for chack git status
git status

//for ignore file or folder (open .gitignore file and type fileName or folderName/ )
touch .gitignore

//for add git one file 
git add filename.html

//for add git all file
git add -A

//git commit with "message"
git commit -a -m "Message here"

//git branch making
git branch branchName

//switch branch 
git checkout branchName

//merge branches (have to shich master Branch)
git merge branchName

//push git 
git push -u origin branchName
